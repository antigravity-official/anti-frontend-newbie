# 안티그래비티 프론트엔드 입사자 기술과제

#### 필수 수행 항목

- [ ] any를 사용하지 말고 Type을 지정
- [ ] 컴포넌트 분리하고, 적당한 폴더 만들어 파일 배치
- [x] 유로화 입력할 때마다 환율 반영하여 원화로 표시
- [x] 환율 정보 로딩중임을 사용자가 알 수 있도록 UI 추가
- [x] 금액 세자리 마다 콤마(,) 를 표시
- [x] 유로화 소수점 2자리까지 표시
- [x] 원화 소수점 표시하지 않음
- [ ] 비니지스 로직을 분리하고 MVVM 구현패턴을 적용

#### 옵션 수행 항목

- [ ] 비효율적으로 작성된 코드가 있다면 변경
- [ ] 테스트 코드를 작성
